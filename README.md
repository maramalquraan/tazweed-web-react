# tazweed-web-react!
Simple web app to give a chance for sellers to add their available time to meet the buyers, also to approve/reject buyer requests.

## Usage
> Signup or Login
> At home, the seller find all the requests from the buyers.
> The seller can approve/reject the requests.
> The seller can use "Add Available Time" button to add a time the buyer can book to meet the seller.
